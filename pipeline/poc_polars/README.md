# PoC Polars Pipeline

Proof of concept for replacing DBT transformations with Polars-based Python scripts.

Implements **v1** of the data-inclusion schema.

## Quick Start

```bash
pip install -e ../../datawarehouse/processings
pip install -e .
python -m poc_polars.run
```

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed setup and usage instructions.
